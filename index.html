Troubleshooting Guide

Battery API Not Supported

  • This feature requires a modern browser (Chrome, Edge, Opera)
  • Firefox has limited support
  • Safari does not support this API
  • Try using Chrome or Edge for best compatibility

Voice Alerts Not Working

  • Ensure your browser allows audio playback
  • Check if your device's volume is turned up
  • Some browsers require user interaction before playing audio
  • Try clicking the "Test Voice Alert" button

Inaccurate Battery Readings

  • Battery API may not be available on all devices
  • Some laptops may show cached battery information
  • Try refreshing the page or restarting your browser
